Task Manager Frontend

Статические файлы пока не собраны из npm-пакета @hexlet/java-task-manager-frontend. Чтобы развернуть полноценный интерфейс, выполните команды ниже из корня проекта после получения доступа к npm-репозиторию:

  1. npm install @hexlet/java-task-manager-frontend
  2. npx build-frontend

После сборки файлы фронтенда автоматически окажутся в src/main/resources/static, и главная страница будет отдавать готовое приложение.

Бэкенд по-прежнему доступен по API-маршрутам. Вы можете работать с ними напрямую через curl или другой HTTP-клиент.
